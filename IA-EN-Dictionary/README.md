Cuis-IA-EN-Dictionary
=====================

Interlingua &lt;--> English lookup window

Tested in Cuis 4.2 rev 3319

To load the package
````Smalltalk
	Feature require: #'IA-EN-Dictionary'
````

Then execute
````Smalltalk
    IEDictWindow open.
````

Or use the World menu->Open command
